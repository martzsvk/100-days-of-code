# Day 10: Calculator Project 🧮

Simple CLI calculator project that performs basic mathematic operations. 

## What I Learned 🛠️
* **Functions with Outputs:** Understanding how functions work with `return` keyword.
* **Recursion:** Implementing a function that calls itself to reset the calculator for a new start without exiting the program.

## How to Run 🚀
1. Copy the code
2. Run `main.py` in your IDE/Terminal
3. Enter your first number, pick an operatin and do math (or meth)
