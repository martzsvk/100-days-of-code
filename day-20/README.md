# Day 20 & 21: The Snake Game 🐍

The classic arcade game built using Python's `turtle` module. This project was a major milestone in understanding Object-Oriented Programming (OOP), specifically Class Inheritance.

## What I Learned 🛠️
* **Class Inheritance:** Creating a `Food` class and a `Scoreboard` class that inherit from the `Turtle` class to gain all its movement and drawing capabilities.
* **Game Physics:** Managing the snake's body segments so they follow the head's path perfectly using a `while` loop and coordinate updates.
* **Data Persistence:** Learning how to keep track of the High Score even after the program restarts.

## Controls
* W / A / S / D - to control the snake
* Collect the food to grow longer and increase your score
* Game Over if you hit the wall or yourself (The high score will be updated if you beat it)
