# Day 23: Turtle Crossing 🐢

### Last Project with `turtle`!!!

## What I Learned 🛠️
* **Car Manager Class:** Implementing a system that spawns "car" objects at random intervals and moves them across the screen.
* **Leveling System:** Increasing the speed of the cars every time the player successfully crosses the finish line.
* **Collision Detection:** Refined logic to detect when any part of a car hits the turtle.

# Controls 🕹️
* `W` key to move the turle forward
* Don't get hit by the car
* There are 10 levels so good luck
