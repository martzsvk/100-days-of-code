# Day 28: Pomodoro Timer 🍅

A functional Pomodoro Productivity App built with Python and Tkinter.

## What I Learned 🛠️
* **Canvas Widget:** Learning to overlay text on top of images (the classic tomato icon) to create a custom UI.
* **Dynamic UI Updates:** Updating labels, text, and labels programmatically (changing the "checkmarks" when a session is finished).
* **Game/App Logic:** Managing the state of the timer (Working vs. Short Break vs. Long Break) and its transition logic.

## How it Works ⚙️
1. Click "Start" to begin a 25-minute work session.
2. The timer counts down in real-time.
3. Once the session ends, it automatically cycles through breaks.
4. Checkmarks appear on the screen to track completed work sessions.

# Requirements 📦
* None (Tkinter is a standard library in Python)
