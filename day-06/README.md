# Day 6: Escaping The Maze 🤖

This project was part of the **100 Days of Code** Python course. The goal was to write a logic-based algorithm for a robot to find the exit in randomly generated mazes.

## What I Learned 🛠️
* **Functions:** Learned how to write code I can reuse as many times I want.
* **While Loops:** Learned about that there is not only a `for` loop (if I want to iterate through something) but also `while` loop if I want to do something until I say otherwise.

## Note on Code 📄
The provided `task.py` is only an exercise where I tried to create a simple function. The final project was about solving mazes on the Reeborg's World site. Since it's browser-based, the code was tested directly on the site.
