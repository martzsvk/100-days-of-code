# Day 45 & 47: Beautiful Soup 📝

In these projects, I moved from fetching data via APIs to extracting it directly from the source code of websites.

## Key Skills 🛠️
* Navigating the HTML DOM tree, using CSS selectors to find specific data, and cleaning raw web text.

* **Project from day 45:** Scraped a list of "100 Greatest Movies" and saved them into a local text file.
* **Project from day 47:** Built an Amazon Price Tracker that monitors a specific product and sends an email alert when the price drops below a threshold.
