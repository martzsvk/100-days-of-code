# Day 19: Turtle Racing Game 🏁

A fun, interactive betting game where multiple "Turtle" instances race across the screen. This project introduces Event Listeners and the concept of Object Instances.

## What I Learned 🛠️
* **Object Instances:** Creating multiple independent objects from the same class (6 different colored turtles), each with its own state (position, color).
* **Coordinate Systems:** Setting up the screen width and height to define the "finish line."
* **State Management:** Tracking which turtle won and comparing it with the user's bet.

## Game Rules 🕹️
1. The console asks: "Which turtle will win the race? Enter the color of F1 Team:"
  * Ferrari (red)
  * McLaren (orange)
  * Mercedes (gray)
  * Sauber (green)
  * Williams (blue)
  * Red Bull (purple)
2. Six turtles line up at the start
3. They move forward by a random distance in each "tick"
4. The first turtle to cross the finish line is the winner
5. The program tells you if you won or lost your bet

## How to Run 🚀
1. Run `main.py`
2. Enter your bet in the pop-up window
3. Enjoy your own F1
