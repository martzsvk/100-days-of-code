# Day 25: U.S. States Game & Pandas Analysis 🗺️

A geographical quiz game that uses the Pandas library to handle data from a CSV file.

## What I Learned 🛠️
* **The Pandas Library:** My first introduction to the most powerful data analysis tool in Python.
* **CSV Manipulation:** Reading data from a `.csv` file and filtering it (e.g., getting coordinates for a specific state).
* **State Management:** Creating a list of "Missing States" at the end of the game and exporting it to a new CSV file (`states_to_learn.csv`) using `.to_csv()`.

### This project requires to install the Pandas library:
* You can do that by typing this into your IDE console
* `pip install pandas`
