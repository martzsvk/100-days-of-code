# Days 41-44: Web Development Foundation 🌐

This section marks my transition from pure Python logic to Frontend Development. Understanding how the web is structured (HTML) and styled (CSS) is the essential bridge to Web Scraping and Full-stack Web Apps.

## What I Learned 🛠️
### HTML5 🏗️
* **HTML:** Using tags like `<header>`, `<p>`, `<div>`.
* **Media Integration:** Working with `<img>`, `<a>` (hyperlinks), and nested lists to organize content.
* **Document Hierarchy:** Mastering the parent-child relationship between elements (the DOM tree).

### CSS3 🎨
* **Selectors & Specificity:** Learning how to target specific elements using `ids`, `classes`, and element types.
* **Typography:** Integrating Google Fonts and managing font-weights, sizes, and line-heights for readability.
