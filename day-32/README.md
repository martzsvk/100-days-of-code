# Day 32: Birthday Wisher 📧

A Python script that monitors a CSV file for birthdays and automatically sends a personalized email to the person on their special day.

## What I Learned 🛠️
* **`smtplib` (Simple Mail Transfer Protocol):** Understanding how Python connects to mail servers (like Gmail or Outlook) to send messages.
* **`datetime` Module:** Learning how to extract the current year, month, and day of the week to trigger specific events.
* **Security & App Passwords:** Implementing secure login methods without hardcoding main account passwords (using Google's App Passwords).

## Requirements 📦
* smtplib & datetime (Standard Library)
* Pandas (External Library)

## Security Note ⚠️
For security reasons, I have replaced my actual email and App Password with placeholders. If you want to run this, you'll need to set up your own App Password in your Google Account settings.
