# Day 18: Turtle Graphics & Hirst Painting 🎨

Exploration of Python's `turtle` module to understand Graphical User Interfaces (GUI), coordinate systems, and random path generation. The final goal was to replicate a famous Damien Hirst "Spot Painting."

## What I Learned 🛠️
* **Turtle GUI:** Controlling a virtual "turtle" to draw shapes like squares, pentagons, and circles.
* **Random Walk:** Implementing a random movement algorithm with randomized RGB colors.
* **Tuple Manipulation:** Understanding how to store and use RGB values in tuples `(R, G, B)`.
* **Color Extraction:** Using the colorgram package to extract a color palette from an actual image.

## The Hirst Painting 🖌️
The script extracts colors from a real painting and uses the Turtle to recreate a grid of colored dots, demonstrating how code can generate art.

## How to Run 🚀
1. You need to install the colorgram library through `pip install colorgram.py` or if you are using PyCharm you can just click on the lamp and install.
2. If you will use the image in the folder you just run `main.py`
3. If you want to change the picture you have to uncomment the code and change the name in `.extract()`
4. Watch the turtle make art
