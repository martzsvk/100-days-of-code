# Day 7: Hangman Game 🪓

This is a classic Hangman game where you have to guess a secret word letter by letter before the man is fully drawn.

## What I Learned 🛠️
* **While & For Loops:** It was more of a revision than learning because I already learned about loops in previous days.
* **Lists & Strings:** Bit of revision but also learning how to display and update the secret word (e.g. `_ _ _ _ _` to `P _ T _ O N`)
* **ASCII Art:** Learned how use multi-line strings to draw the different stages of hangman.

---

### How to Play 🚀
1. Copy the code
2. Run `main.py` on your terminal
3. Type a letter and **Enter**
4. Enjoy & don't let the man die
